Note bài cũ:
API là tên của hàm chức năng(gần giống) khi front-end gọi API thì sever sẽ có hàm API tương 
ứng để giải quyết.
Một số http menthod gởi phần body: post, put, patch

 bài mới:
-    port dùng để xác định các dịch vụ(service)(0-6535)
-    ip (internet protoco) để xác định các máy(thiết bị...)
    status code:
              1XX   Yêu cầu đang được xử lí 		    100 Continue
              2XX   Xử lí thành công       		    200 OK – Thành công
              3XX   Chuyển hướng (304Not Modified)          301 Moved Permanently – Chuyển hướng vĩnh viễn
	      4xx   Lỗi phía client           		    400 Bad Request – Request sai cú pháp
	      5xx   Lỗi phía sever       	            500 Internal Server Error – Lỗi server
-    Nodejs là một môi trường chạy javascript
-    Npm (note package manage) là một công cụ quản lí các 
    thư viện cho js
